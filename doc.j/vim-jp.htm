目的

このメーリングリストは，vi クローン「vim」の日本語のメーリングリストです。開発者およびユーザ向けのものであり，質問 / バグレポート / 将来の機能 などについて論議するものです。

なお，管理は Kazuhiro Sakai ( vim-jp-admin@sandalwood.net ) が行っております。

参加

vim-jp-ctl@sandalwood.net 宛に次の行だけを 本文 に記述したメールを送ってください。

　　　　subscribe "あなたの名前"

たとえば私 ( Kazuhiro Sakai ) が参加するには，次のようになります。

　　　　subscribe Kazuhiro Sakai

メールを送ると，次のような確認メッセージが返信されます。

　　　　confirm 199907111739315060045 Kazuhiro Sakai
　　　　......
　　　　......
　　　　......
返信されたメールの "confirm" で始まる行を vim-jp-ctl@sandalwood.net 宛に返信してください。登録が完了すると Welcome メッセージが返信されます。

投稿

vim-jp@sandalwood.net 宛にメールを送ってください。

脱退

vim-jp-ctl@sandalwood.net 宛に次の行だけを 本文 に記述したメールを送ってください。

　　　　# unsubscribe

この時，メーリングリストに入会したメールアドレスから送る必要がありますので注意してください。

ガイドライン

vim-jp-ctl@sandalwood.net 宛に次の行だけを 本文 に記述したメールを送ってください。

　　　　# guide
